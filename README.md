# Calculadora de Honorários

Ferramenta simples para calcular honorários advocatícios com base no valor da causa.

## Como usar

1. Informe o valor da causa
2. Selecione o tipo de ação
3. O sistema calculará o honorário sugerido conforme a tabela da OAB

## Fórmula

O honorário mínimo é de **10%** sobre o valor da causa, conforme o Código de Ética da OAB.

## Instalação

```bash
python calcular.py
```
